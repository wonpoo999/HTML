HTML 
HTML 프론트엔드는 사용자가 직접 보는 화면과 상호작용하는 부분을 담당합니다. 웹사이트의 디자인, 버튼, 메뉴, 애니메이션 등을 구현하며, 사용자 경험(UX)과 인터페이스(UI)를 최적화하는 역할을 합니다.
HTML ●    주요 프로그래밍 언어 및 기술:
HTML ○    HTML: 웹 페이지의 구조를 정의하는 마크업 언어
HTML ○    CSS: 웹 페이지의 스타일을 지정하는 스타일 시트 언어
HTML ○    JavaScript: 웹 페이지의 동적 기능을 구현하는 프로그래밍 언어
HTML ○    프레임워크 및 라이브러리 : React, Vue.js, Angular 등
HTML 백엔드는 사용자가 직접 볼 수 없는 서버, 데이터베이스, 애플리케이션 로직을 관리하는 부분입니다. 사용자의 요청을 처리하고 데이터를 저장 및 관리하며, 보안과 성능 최적화를 담당합니다.
HTML ●    주요 프로그래밍 언어 및 기술:
HTML ○    Java: 안정성과 확장성이 뛰어난 백엔드 언어
HTML ○    Python: 데이터 처리 및 AI 개발에도 활용되는 범용 언어
HTML ○    Node.js: 서버 기능을 구현 할 수 있는 자바스크립트 백엔드 개발 플랫폼
HTML ○    데이터베이스 : 오라클, MySQL, PostgreSQL, MongoDB 등
HTML 프론트엔드와 백엔드는 서로 협력하여 웹사이트와 애플리케이션이 원활하게 작동하도록 합니다.
