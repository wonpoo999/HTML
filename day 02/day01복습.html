Definition of CSS

CSS(Cascading Style Sheets, 종속형 스타일 시트)



웹 페이지의 스타일을 지정하는 언어

CSS(Cascading Style Sheets, 종속형 스타일 시트)는 웹 페이지의 스타일과 디자인을 담당하는 언어입니다. HTML이 웹 페이지의 구조(내용)를 담당한다면, CSS는 그 구조에 디자인과 레이아웃을 입히는 역할을 합니다. 글씨 색, 크기, 베경, 여백, 정렬 등을 바꿀 수 있습니다

🔧 CSS의 주요 기능

기능 성명
  • 색상 지정 글자색, 배경색 등 지정 가능
  • 폰트 설정 글꼴, 크기, 굵기 등 지정
  • 배치 조정 요소 위치 지정 (정렬, 정렬 방향 등)
  • 크기 조절 너비, 높이 등 조정
  • 애니메이션 간단한 전환, 애니메이션 효과 구현
  • 반응형 디자인 화면 크기에 따라 스타일 변경 가능 (미디어 쿼리 사용)
