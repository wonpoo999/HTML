<프론트 엔드> 용어 정리

  1. HTML
    • 웹 페이지 뼈대를 구성하는 마크업 언어
    • 제목, 문단, 이미지 등의 요서 정의 ≶h1>, ≶p>,≶img>
  2. CSS
    • HTML 요소에 색상, 크기, 레이아웃 등 스타일 지정
    • 반응형 웹, 애니메이션 등 시각 효과 적용 가능
  3. JavaScript
    • 페이지 동작을 구현하는 프로그래밍 언어
    • 클릭, 입력 등 사용자 이벤트 처리 가능
  4. DOM(Document Object Model)
    • HTML을 트리 구조로 표현한 객체 모델
    • JavaScript로 HTML 요소를 조작하거나 감지 가능
  5. Website
    • 인터넷을 통해 접근 가능한 웹 페이지들의 모음
    • HTML, CSS, JS 등으로 구성
    • 도메인(URL)과 서버가 필요함
  6. HTTP /HTTPS
    • 웹에서 테이터를 주고받는 규약(protocol)
    • HTTP: HyperText Transfer Protocol
    • HTTPS: 보안이 적용된 HTTP (SSL 인증서 사용, 자물쇠 아이콘)
